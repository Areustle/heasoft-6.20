NAME

uvotskyfield - build a UVOT SKY image

USAGE

uvotskyfield outfile=<filename> catspec=<filename>

DESCRIPTION

This tool creates a sky image based on star catalog information.

PARAMETERS

outfile [filename]
Name for output file (FITS image).

catspec [filename]
Star catalog descriptor.

ra [real]
Right ascension at center of output image [deg].

dec [real]
Declination at center of output image [deg].

arcmin [real]
Width/height of output image [arcmin].

fwhm [real]
PSF/FWHM [pixels].

zero [real]
Zero point for converting magnitudes to counts.

(clobber = no) [boolean]
Standard HEAdas clobber parameter.

(chatter = 1) [enumerated integer]
Standard HEAdas chatter parameter. A range of 0 to 5 is enforced by the min and max fields in the parameter file.
