NAME

uvotattcorr1 - back-end for uvotattcorr

USAGE

uvotattcorr1 infile=<sw12345678012sat.fits> deltafile=<delta.fits> outfile=<uvotatt.fits>

DESCRIPTION

This tool is the back-end for uvotattcorr. uvotattcorr1 performs the quaternion multiplications and FITS i/o as set up by uvotattcorr.

It is not recommended or expected that users will run this tool directly since uvotattcorr provides an interface for use with existing products.

PARAMETERS

infile [filename]
Swift attitude file (normally sw<obsid>sat.fits).

deltafile [filename]
Attitude file with records corresponding to UVOT aspect corrections.

outfile [filename]
Attitude file corrected for UVOT.

(alignfile = NONE) [filename]
Alignment file. The special value NONE indicates to not create this output. The special value CALDB causes the alignment file to be retrieved from the CALDB based on the header of infile.

(clobber = no) [boolean]
Overwrite existing files?

(history = yes) [boolean]
Write HISTORY keywords to output?

(chatter = 3) [integer] <1-5>
Standard HEAdas chatter parameter.
