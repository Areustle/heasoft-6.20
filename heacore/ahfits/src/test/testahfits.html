NAME

testahfits - Test ahfits library code.

USAGE

testahfits

DESCRIPTION

This is the unit test of the ahfits library. It reads as input a modified Suzaku data file (see ahdemo) and writes another file of the same format, but with the content in the ASIC_MOCKUP column determined from the SEGMENT column. Thus the output file it produces is correct as the input file for ahdemo's unit test.

PARAMETERS

None.

USAGE

This tool must be run from the directory headas/astroh/gen/lib/ahfits/src/test/ in order to find the necessary input file(s). 

      testahfits
