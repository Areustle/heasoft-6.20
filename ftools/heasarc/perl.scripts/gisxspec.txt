<h2>NAME</h2>

gisxspec - Do spectral analysis for gis data.

<h2>USAGE</h2>

<tt>gisxspec  g2elist g3elist  xcol ycol ra dec radius bin srcregfile backregfile srcphafile backphafile
srcarffile</tt>

<h2>DESCRIPTION</h2>

'gisxspec' Do spectral analysis for gis data.
<p>

<h2>PARAMETERS</h2>
<dl>

<dt>g2elist [string]
<dd>A list of the G2 event file names. The list items should be delimited by blank
space.
<p>

<dt>g3elist [string]
<dd>A list of the G3 event file names. The list items should be delimited by blank
space.
<p>

<dt>xcol [string]
<dd> X coordination column of the FITS events file.
<p>
<dt>ycol [string]
<dd> Y coordination column of the FITS events file.
<p>

<dt>ra ="def" [string]
<dd>Right ascension (in h:m:s) of the center of the default selected area. If ra = "def",  the ra 
is the same as that of the center of the image. 
<p>

<dt>dec ="def" [string]
<dd>Declination (in degree:m:s) of the center of default selected area. If dec = "def",  the dec 
is the same as that of the center of the image. 
<p>

<dt>radius ="def" [string]
<dd>The radius (arcmin) of the default selected area. If dec = "def",  the radius 
is 1/8 of the height of the image. 
<p>

<dt>bin =15 [integer]
<dd>bin size
<p>

<dt>srcreg [filename]
<dd>File name of the source region file.
<p>

<dt>backreg [filename]
<dd>File name of the background region file.
<p>

<dt>srcpha [filename]
<dd>File name of the combined g2 and g3 source pha file.
<p>

<dt>backpha [filename]
<dd>File name of the combined g2 and g3 background pha file.
<p>

<dt>srcarf [filename]
<dd>File name of the combined g2 and g3 source arf file.
<p>

<dt>(g2srcpha) [filename]
<dd>File name of the  g2 source pha file.
<p>

<dt>(g2backpha) [filename]
<dd>File name of the g2 background pha file.
<p>

<dt>(g2srcarf) [filename]
<dd>File name of the g2 source arf file.
<p>

<dt>(g3srcpha) [filename]
<dd>File name of the g3 source pha file.
<p>

<dt>(g3backpha) [filename]
<dd>File name of the g3 background pha file.

<dt>(g3srcarf) [filename]
<dd>File name of the g3 source arf file.
<p>

<dt>(device ="/xw") [device]
<dd>xspec plot device.
<p>

<dt>(timeout =200) [double]
<dd>The time (in second) for access to POWplot. It will stop if time pass over
"timeout" but it still cannot connect to POWplot.
<p>


<dt>(chatter = 1) [integer]
<dd>Controls the verbosity level of the output.  Currently
this parameter has no effect.
<p>
</dl>
