    
TKEXOMERGE
	
USAGE

    tkexomerge -i catfile -o outfile 

        or

    tkexomerge 
	    
DESCRIPTION

 This script allows to select, via a Tcl/Tk GUI, event files from the 
 catalog file associated to each EXOSAT LE observation. The selected event 
 files are than merged by running the task 'exomerge'.  

FLAGS:

    The following flags require arguments. If these paramaters are 
    not given, pop-up windows will appear to enter filenames. 

     -i - input catalog file 
     -o - output merged event file
