Usage: add_sis_rmf evtfile addrmf_file caldb_dir 
Reports what percentage of the total counts in evtfile are in each chip 
If addrmf_file is given, an input file for addrmf is created 
If caldb_dir is not given, /FTP/caldb/data/asca/sis/cpf/94nov9/ is used 
Filenames put into addrmf_file are: 
s?c?g0234p40e1_512v0_8i.rmf 
SIS # is determined from evtfile
